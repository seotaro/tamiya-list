# tamiya-list

ウェブサイトをスクレイピングしてタミヤ模型の塗料一覧をJSON形式とCSV形式で出力する。

## Run

「アクリル塗料ミニ」

```bash
node listup-acrylic-paint-mini.js
```

「エナメル塗料」

```bash
node listup-enamel-paint.js
```

「ラッカー塗料」

```bash
node listup-lacquer-paint.js
```
